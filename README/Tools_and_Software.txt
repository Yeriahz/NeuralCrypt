# Tools and Software for Crypto AI Project

**Development Environment:**
- Python: https://www.python.org/
- Jupyter Notebook (via Anaconda): https://www.anaconda.com/
- VS Code: https://code.visualstudio.com/

**APIs and Data Sources:**
- CoinGecko API: https://www.coingecko.com/
- CryptoCompare API: https://www.cryptocompare.com/

**Libraries and Frameworks:**
- TensorFlow: https://www.tensorflow.org/
- PyTorch: https://pytorch.org/
- Pandas, NumPy: Installed via `pip install pandas numpy`
- Matplotlib, Plotly: Installed via `pip install matplotlib plotly`
- BeautifulSoup, Selenium: Installed via `pip install beautifulsoup4 selenium`
- Hugging Face Transformers: Installed via `pip install transformers`

**Cloud and Deployment:**
- AWS CLI: https://aws.amazon.com/cli/
- Google Cloud SDK: https://cloud.google.com/sdk
- Docker: https://www.docker.com/

**Version Control and Collaboration:**
- Git: https://git-scm.com/
- GitHub Desktop: https://desktop.github.com/
